# Personal Expense Tracker (Python)

A command-line based Expense Tracker built using Python to manage and analyze daily expenses.

---

## Features

- Add new expenses with date, category, amount, and note  
- View all recorded expenses  
- Filter expenses by category  
- Calculate total spending  
- Generate monthly reports  
- Persistent data storage using JSON  
- Optional data visualization using matplotlib  

---

## Technologies Used

- Python  
- JSON  
- datetime  
- os  
- matplotlib (optional)  

---

## Project Structure

expense-tracker/

├── storage.py     # File handling (load/save)
