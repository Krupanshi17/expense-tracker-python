# Personal Expense Tracker (Python)

A command-line based expense tracker built using Python.

#Features
- Add expenses with date, category, amount, and note
- View all expenses
- Filter expenses by category
- Calculate total spending
- Generate monthly reports
- Data stored using JSON
- Optional visualization using matplotlib

#Technologies Used
- Python
- JSON
- datetime
- os
- matplotlib (optional)

## How to Run
```bash
python3 main.py

    -storage.py → File handling

    -reports.py → Reports & analysis
